# 幹你JS

## Project setup
```
npm install
```

### 開發環境
```
npm run serve
```

### 產生正式環境
```
npm run build
```

### 部署到 Firebase Hosting
```
npm run deploy
```

### 產生正式環境並部署到 Firebase Hosting
```
npm run buildDeploy
```

### Lints and fixes files
```
npm run lint
```

See [Configuration Reference](https://cli.vuejs.org/config/).
